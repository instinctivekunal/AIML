The skewness can be calculated by the skew() of dataframe. Following are the cases
if > 0 then it is an asymmetrical distribution and the tail is left skewed
if < 0 then it is an asymmetrical distribution and the tail is right skewed
if = 0 then it is a symmetrical distribution

In our case, all the columns are left skewed based on the output values. And age column is very close to a symmetrical distribution

a. Do charges of people who smoke differ significantly from the people who don't?¶

In [306]:
# T-test to check dependency of smoking on charges
# Stating the Null Hypothesis
Ho = "Charges of smoker and non-smoker are same"   
# Stating the Alternate Hypothesis
Ha = "Charges of smoker and non-smoker are not the same"   

# Selecting charges corresponding to smokers as an array
x = np.array(data[data.smoker == 'yes'].charges)
# Selecting charges corresponding to non-smokers as an array
y = np.array(data[data.smoker == 'no'].charges) 

#Performing an Independent t-test
t, p_value  = stats.ttest_ind(x, y, axis = 0)  
p_value
# Setting our significance level at 5%
if p_value < 0.05:  
    print(f'{Ha} as the p_value ({p_value}) < 0.05')
else:
    print(f'{Ho} as the p_value ({p_value}) > 0.05')

b. Does bmi of males differ significantly from that of females?¶

In [307]:
# T-test to check dependency of bmi on gender
# Stating the Null Hypothesis
Ho = "Gender has no effect on bmi"   
# Stating the Alternate Hypothesis
Ha = "Gender has an effect on bmi"   

# Selecting bmi values corresponding to males as an array
x = np.array(data[data.sex == 'male'].bmi)
# Selecting bmi values corresponding to females as an array
y = np.array(data[data.sex == 'female'].bmi) 

#Performing an Independent t-test
t, p_value  = stats.ttest_ind(x,y, axis = 0)  

# Setting our significance level at 5%
if p_value < 0.05:  
    print(f'{Ha} as the p_value ({p_value.round()}) < 0.05')
else:
    print(f'{Ho} as the p_value ({p_value.round(3)}) > 0.05')

Gender has no effect on bmi as the p_value (0.09) > 0.05

# Chi_square test to check if smoking habits are different for different genders
# Stating the Null Hypothesis
Ho = "Gender has no effect on smoking habits"
# Stating the Alternate Hypothesis
Ha = "Gender has an effect on smoking habits"   

# Contingency table of sex and smoker attributes
crosstab = pd.crosstab(data['sex'], data['smoker'])  

#Performing the Chi square test
chi, p_value, dof, expected =  stats.chi2_contingency(crosstab)

# Setting our significance level at 5%
if p_value < 0.05:

# Test to see if the distributions of bmi values for females having different number of children, are significantly different
# Stating the Null Hypothesis
Ho = "No. of children has no effect on bmi"
# Stating the Alternate Hypothesis
Ha = "No. of children has an effect on bmi"   

female_df = copy.deepcopy(data[data['sex'] == 'female'])

zero = female_df[female_df.children == 0]['bmi']
one = female_df[female_df.children == 1]['bmi']
two = female_df[female_df.children == 2]['bmi']

#Performing the one-way ANOVA test
f_stat, p_value = stats.f_oneway(zero,one,two)

# Setting our significance level at 5%
if p_value < 0.05: 
    print(f'{Ha} as the p_value ({p_value.round(3)}) < 0.05')
else:
    print(f'{Ho} as the p_value ({p_value.round(3)}) > 0.05')
